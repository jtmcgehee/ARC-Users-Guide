# Step 1: Project Information

This is the first screen you see when starting a new project. Enter the basic details that identify your project and site â€” this information appears in the headers of generated reports.

## Project Details

| Field | Required | Description |
|---|---|---|
| **Client Name** | Yes | The client or organization commissioning the assessment. |
| **Project Number** | No | Your internal project or task number. |
| **Site Name** | Yes | The name of the site. Also used as the suggested filename when saving. |

## Site Location

| Field | Required | Description |
|---|---|---|
| **Street Address** | Yes | Street address of the site. |
| **City** | Yes | City where the site is located. |
| **State** | Yes | State where the site is located. |
| **ZIP Code** | Yes | ZIP code for the site. |

**Date Created** and **Last Modified** timestamps are displayed at the bottom and updated automatically.

All required fields must be completed before you can generate reports. Changes are saved automatically.
